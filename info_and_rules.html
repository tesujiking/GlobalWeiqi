📚 赛事规则与说明

了解全球围棋超级联赛的各项规章制度和运作方式

📋 赛事章程

第一章 赛事宗旨

  1. 普及推广围棋运动，提供全球围棋爱好者交流平台。
  2. 提供专业、公正的赛事环境，促进选手棋艺进步。
  3. 建立公开透明的等级分系统，客观反映选手实力。

第二章 赛事组织

  1. 主办单位：全球围棋超级联赛组委会。
  2. 承办单位：全球围棋超级联赛组委会。

第三章 参赛资格

  1. 所有热爱围棋的个人，不分国籍、年龄，均可报名参赛。
  2. 首次参赛选手需进行等级水平认定或提交现有水平证明。
  3. 选手须遵守赛事所有规定，维护比赛公平公正。

第四章 比赛规则

  1. 比赛采用全球围棋超级联赛规则。
  2. 选手在比赛群里自行约棋，采用野狐平台对局，所有对局均尽可能为分先对局。但也会有让先，让子对局。
  3. 比赛用时：根据所在赛区不同，每方20/30/40分钟，3次30/60秒读秒。
  4. 贴目：黑贴3又3/4子（7.5目）。

第五章 等级分系统

  1. 本联赛采用世界先进的等级分系统进行选手实力评估和排名。
  2. 所有参赛选手根据初次报名参赛级别定不同的初始等级分。
  3. 每场比赛结束后，根据胜负结果和选手间等级分差异，更新双方等级分。
  4. 等级分计算可以本站参考等级分计算器工具

第六章 违规与处罚

  1. 严禁作弊行为，包括但不限于使用AI辅助、代下、串通作弊等。
  2. 任何被发现的作弊行为将有可能导致立即取消比赛资格，永久禁赛，并公示名单等。
  3. 选手在比赛中须尊重对手，文明对弈。

第七章 附则

  1. 本章程的最终解释权归全球围棋超级联赛组委会所有。
  2. 未尽事宜，组委会将另行通知或补充规定。

🎲 麦克马洪赛制

麦克马洪制（McMahon System）是瑞士制的一种变体，常用于围棋比赛。它旨在解决传统瑞士制中，强手在初期就遇到弱手，导致强手难以通过胜利快速提升排名的问题。

核心特点：

  1. 初始麦克马洪分 (MM Score)：
    • 参赛选手根据其预先认定的棋力（如段位或等级分）被赋予一个初始分数。
    • 强手有较高的初始麦克马洪分，弱手有较低的初始分。
    • 例如：5D初始分为34分，1K为29分，25K为5分。
  2. 积分晋级：
    • 每赢一盘棋，选手在原有的麦克马洪分基础上增加1分。
    • 输棋不扣分，和棋双方各加0.5分，未参赛加0.5分。
  3. 配对原则：
    • 第一轮：通常将所有选手分为若干个“麦克马洪段位组”。同组内的选手随机配对，或根据初始分高低进行配对，避免强手一开始就遭遇弱手。
    • 后续轮次：在同一轮中，配对的原则是寻找麦克马洪分相同或相近的对手。
    • 避免两位选手在同一赛事中对弈两次。
  4. 最终排名：
    • 比赛结束后，选手的最终排名依据其累积的麦克马洪总分（初始分 + 胜局分）进行排序。
    • 若总分相同，则通常根据小分（对手分SOS），小小分（对手的对手分SOSOS）来决定名次。

麦克马洪制的优势：

  • 初期比赛质量高：强手之间、弱手之间在早期就能相遇，减少了实力悬殊的比赛。
  • 排名更合理：能更准确地体现选手的真实实力，因为强手必须战胜同级别或更强的对手才能获得高分。
  • 鼓励参与：即使输掉几盘棋，选手也能根据初始分保持相对较高的排名，不至于过早失去竞争力。

在全球围棋超级联赛中，麦克马洪赛制结合等级分系统，旨在为所有参赛者提供公平、刺激且能充分展示实力的比赛体验。

🔄 配对流程

本联赛采用**改良麦克马洪赛制**进行配对，以确保每轮对局的公平性与竞技性。以下是详细的配对流程：

  1. 初始轮次配对：
    • 所有选手根据其初始等级分（或预设麦克马洪分）进行分组。
    • 第一轮配对将在同一等级分段内随机进行。
    • 配对时会尽量避免同队选手相遇。
  2. 后续轮次配对（麦克马洪分优先）：
    • 从第二轮开始，系统将优先配对当前麦克马洪分（MM Score）相同的选手。
    • 若麦克马洪分相同的选手不足或已匹配，则会寻找分值相近的选手。
    • 配对时，会尽量避免：
      • 已对弈过的选手再次相遇。
      • 同队选手相遇。
      • 色差过大的配对（即一方执黑过多，一方执白过多）。
  3. 棋份与贴目：
    • 根据比赛的棋份规则而定，大多数对局均采用分先对局。
    • **黑方贴目**：中国规则，黑贴3又3/4子（7.5目）。
    • **让先对局黑181胜，让2子或更多，黑贴还让子数的一半，和棋判为白胜。
  4. 配对结果公布：
    • 每轮配对结果将在规定时间内公布于网站。
    • 选手需在规定时间内完成对局，并提交赛果。
  5. 特殊情况处理：
    • 若有选手迟到、弃权或退赛，系统将根据规定进行处理。
    • 对局过程中出现争议，由裁判组进行裁决。

通过严格的配对流程，我们力求为每位选手提供旗鼓相当的对手，最大化比赛的公平性和乐趣。

💰 保证金制度

为确保比赛的顺利进行和选手的积极参与，全球围棋超级联赛实行保证金制度。

  1. 保证金金额：
    • 每位参赛选手需在报名时缴纳固定金额的保证金（10元人民币）。
  2. 保证金用途：
    • 保证金主要用于约束选手的比赛行为，避免无故弃权、迟到等影响比赛正常进行的情况。
    • 在极少数情况下，保证金也可能用于支付因选手违规行为产生的罚款。
  3. 保证金返还条件：
    • 选手在完成所有规定轮次比赛后，保证金将全额无息返还。
    • 联赛为不间断的常年比赛，只要保证金没有退还，可以随时退出或加入比赛，若决定永久退赛，则在无违规情况下，保证金全额退还
  4. 保证金扣除情况：
    • 无故弃权、严重迟到（超过双方约定时间15分钟），需在比赛群里发一个5元的致歉红包
    • 在比赛中被认定有作弊行为。
    • 严重违反赛事章程或体育道德的行为。
    • 未能按时提交赛果，且未及时与组委会沟通。

    具体扣除金额和比例将由组委会根据违规情节的严重程度裁定。

  5. 退赛与转让：
    • 选手如需退赛，请务必提前与组委会联系，说明情况。
    • 保证金不可转让给其他选手。
  6. 退款流程：
    • 符合返还条件的选手，组委会将会及时退款。
    • 退款将原路返回至缴费账户，请选手留意查收。

保证金制度旨在保障所有选手的公平权益，维护赛事的良好秩序。感谢您的理解与配合。
